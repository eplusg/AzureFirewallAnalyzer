Azure Firewall Rules Analyzer

How to Use

  1. Export your Azure Firewall policy: Save your policy as a JSON file.
  2. Import the JSON file: Click the 'Choose File' button below and select the exported JSON file.
  3. Analyze the policy: The analyzer will display the priority order of each rule.

Disclaimer: No data is stored server-side. All processing occurs in your browser.
